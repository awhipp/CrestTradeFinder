Realtime EVE Trade Finder

How to use this tool

  1. Select which trade hub you will be making purchases from.
  2. Find which item you wish to trade
  3. ISK ratio = BUY_PRICE / PROFIT. Essentially it represents return on investment
  4. "Sell at" references the hub that is buying at a higher price
  5. Make sure to note the volume of items being purchased at the destination
